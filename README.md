# Library

A project done as part of the Odin Project curriculum.

This project is a part of the JavaScript Course. The purpose of this project is to create a library using HTML, CSS, and JavaScript to display a webpage that can store book objects in an array. Using Javascript, a function will take the user's input and store a new book object. Another function will loop through the
array and display eack book on the webpage.

Using HTML and CSS, a button will be created that will bring up a form when clicked. The form allows the user to input the details for the new book: author, title, number of pages, and whether it's read when clicked. There will also be
